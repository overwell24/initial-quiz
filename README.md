# initial-quiz

초성 퀴즈 게임

## 기술 스택

프로그래밍 언어: `Java` <br>
GUI: `SWING` <br>
DATABASE: `MySQL` <br>
IDE: Apache Netbeans 14 <br>

## 프로젝트 구조

mvc 패턴으로 프로젝트를 구성했다.

- controller
- service
- dao
- dto
- frame
- db
